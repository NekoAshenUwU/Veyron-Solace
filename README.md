## Nekoの小金库：做成 App（PWA）

已新增基础 PWA 文件，可安装到手机主屏：

1. 使用任意静态服务器打开项目（示例：`python3 -m http.server 4173`）。
2. 浏览器访问 `http://localhost:4173/index.html`。
3. 在 Chrome/Edge/Safari 选择「添加到主屏幕」即可像 App 一样打开。
4. 记账与评价数据保存在浏览器本地（localStorage）。

> 目前是 **PWA 版本**（轻量 App）。若你要打包成 Android `.apk` 或 iOS 安装包，下一步可接 Capacitor。
